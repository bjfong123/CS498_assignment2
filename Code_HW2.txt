to send GET and POST requests to Node.js endpoints. In order to calculate latency, 

I measured the time difference using time.time() before and after each request, and 

then averaging over 10 times. Code specifics are found in the latency.py file.

in the US, and then querying into the EU list to check if the UIUD is there or not. For code 

specifics, check the consistency. py file.
